#P5B-2 — Besoins & tendances

En te basant sur les sections 3.3 et 2A.3
des 25 fichiers PDATA.

NOTE MÉTHODOLOGIQUE :
Les commentaires analysés proviennent uniquement
des 15 vidéos les plus vues de chaque chaîne,
filtrés par l'algorithme YouTube comme "pertinents".
Ce corpus favorise les contenus anciens et populaires.
Les signaux récents (2024-2025) sont partiellement
visibles via les évolutions thématiques (2A.3)
mais sous-représentés dans les commentaires.
Indique systématiquement le nombre de PDATA
sources pour chaque observation.

---

## P5B-2.1. Ce que les cyclistes cherchent

**Besoins bien couverts (≥3 chaînes) :**
Besoins explicitement satisfaits dans les commentaires.
Format :
- [Besoin] — [citation exacte] — [PDATA source]
  Présent dans : [liste des chaînes]

**Besoins exprimés mais sous-servis (≥2 chaînes) :**
Demandes récurrentes sans réponse claire
dans l'échantillon.
Format :
- [Besoin] — [citation exacte] — [PDATA source]
  Présent dans : [liste des chaînes]

**Frustrations récurrentes (≥2 chaînes) :**
Insatisfactions exprimées dans les commentaires.
Format :
- [Frustration] — [citation exacte] — [PDATA source]
  Présent dans : [liste des chaînes]

**Marques citées spontanément (≥2 chaînes) :**
Basé sur les sections 3.3 Marques/produits
de tous les PDATA.
Format :
- [Marque] — citée dans [N] chaînes
  — [mention exacte — PDATA source]

## P5B-2.2. Signaux thématiques 2022-2025

Basé uniquement sur section 2A.3 des 25 PDATA.
Ces signaux reflètent l'évolution des catalogues
— pas des commentaires.

**Sujets en montée :**
Thèmes qui apparaissent ou s'intensifient
dans les titres 2023-2025 vs 2022.
Format :
- [Sujet] — [titre concret — PDATA source]
  Présent dans : [N chaînes]

**Sujets en déclin :**
Thèmes présents en 2022 qui s'effacent
progressivement des catalogues.
Format :
- [Sujet] — [titre concret — PDATA source]
  Présent dans : [N chaînes]

**Sujets demandés absents du catalogue :**
Demandes explicites dans les commentaires (3.3)
que les chaînes ne couvrent pas encore.
Format :
- [Sujet] — [citation exacte — PDATA source]
