## Structure du dépôt

1. **0-Archive** : Historique et anciennes versions des documents.
2. **1-Donnee_Brute** : Données sources et fichiers bruts (CSV, Excel, etc.).
3. **2-Analyse** : Rapports, schémas techniques et audits.
4. **3-Strategie** : Notes de réflexion, mindmaps et planification.
5. **4-Action** : Scénarios n8n, prompts et ressources opérationnelles.

---

## Nomenclature des fichiers

Chaque fichier est nommé selon la règle suivante :
`Article#_NomDuFichier_Date.Format`

* **Article#** : Numéro de l'article de référence sur le site.
* **Date** : Format AAAA-MM-JJ.
* **Format** : .csv, .pdf, .json, .txt, etc.

*Note : Les dossiers nommés avec le suffixe "_Folder" contiennent des séries de fichiers de même nature pour faciliter la lecture.*
